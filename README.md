# Video-Game-Dashboard
Dashboard Description

This dashboard provides a comprehensive analysis of the global video game market. It visualizes key metrics such as total sales, top publishers, popular genres, and sales trends over time. The dashboard is interactive, allowing users to filter data based on publisher and console to gain deeper insights.

 Tech Stack:-
 
1.Power BI → Used for creating interactive dashboards and visual storytelling

2.Excel/CSV Dataset → Used as the primary data source

3.Data Cleaning (Power Query) → To handle missing values, duplicates, and inconsistent data

4.DAX (Data Analysis Expressions) → For calculating KPIs like total sales, rankings, etc.

These tools help convert raw data into meaningful and interactive business insights.

Business Problem:-

Gaming companies and stakeholders often struggle to:

-Identify which genres are most profitable

-Understand which publishers dominate the market

-Track sales growth over time

-Recognize top-performing games


 Business Impact:-
 
1.Helps companies focus on high-performing genres (like Sports)

2.Identifies market leaders (Activision)

3.Supports strategy by showing growth trends over years

4.Helps in understanding which type of games generate maximum revenue

5.Improves decision-making using data instead of assumptions

Key Insights:-

-Sports genre has the highest sales → High customer demand

-Activision is the leading publisher → Strong market presence

-A few top games contribute majorly to total sales

Walkthough of Key Visuals:-

1.KPI Cards → Show overall performance (Sales, Publisher, Genre, Total Games)

2.Bar Chart (Top Genres) → Identifies most profitable genres

3.Donut Chart (Publisher Share) → Shows market distribution among publishers

4.Line Chart (Year-wise Trend) → Displays sales growth over time

5.Bar Chart (Top Games) → Highlights best-selling games

6.Filters → Allow customized analysis

“I built this dashboard to transform raw gaming data into actionable insights, helping identify top-performing genres, publishers, and sales trends for better business decision-making.”

 Dashboard Goal 
The goal of this dashboard is to analyze global video game sales and present key insights on top genres, publishers, and sales trends in a clear and interactive way. It helps stakeholders understand market performance and make data-driven decisions.
